# LANDING PAGE

A publicly accesible page is shown as the landing page.

**URL** : `https://localhost:10443`

**Method** : `GET`

**Auth required** : No

**Curl example**

```bash
curl -X GET https://localhost:10443
```

## Success Response

**Code** : `200 OK` or `301 Redirect`

**Content example**

redirected to http://rbccps.org/smartcity/doku.php
